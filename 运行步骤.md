在IDEA中运行微服务项目，步骤如下：

1. 打开项目

- 用IDEA打开 e:\code\train 根目录（父POM项目）
- IDEA会自动识别所有子模块
2. 确保Maven依赖已加载

- 右键父项目的 pom.xml → Maven → Reload Project
- 等待依赖下载完成
3. 启动顺序（重要）

先确保外部服务已启动：

**1. MySQL（端口3306）**
- 启动方式：Windows服务自动启动
- 检查服务：`services.msc` → 找到"MySQL80"服务
- 如果未启动，右键 → 启动
- 验证：`mysql -u root -p` 能登录即表示正常

**2. Redis（端口6379）**
- 启动命令：`D:\Redis\redis-server.exe D:\Redis\redis.windows.conf`
- 或者双击 `D:\Redis\redis-server.exe`
- 验证：`D:\Redis\redis-cli.exe ping` 返回 PONG

**3. Nacos（端口8848）**
- 启动命令：`C:\Users\11235\ai-infra\nacos\standalone\nacos-3.2.2\bin\startup.cmd -m standalone`
- 注意：必须在TRAE外部启动（TRAE沙箱无法写入日志）
- 验证：访问 `http://localhost:8848/nacos`（用户名/密码：nacos/nacos）

**4. RocketMQ（端口9876）**
- 先启动NameServer：
  ```powershell
  cd D:\rocketmq\rocketmq-all-5.5.0-bin-release\rocketmq-all-5.5.0-bin-release\bin
  .\mqnamesrv.cmd
  ```
- 再启动Broker（新开窗口）：
  ```powershell
  cd D:\rocketmq\rocketmq-all-5.5.0-bin-release\rocketmq-all-5.5.0-bin-release\bin
  set "ROCKETMQ_HOME=D:\rocketmq\rocketmq-all-5.5.0-bin-release\rocketmq-all-5.5.0-bin-release"
  .\mqbroker.cmd -n localhost:9876 autoCreateTopicEnable=true
  ```
- 注意：必须在TRAE外部启动
- 验证：`netstat -ano | findstr 9876` 能看到监听端口
然后按顺序启动各微服务（右键 Application 类 → Run）：

启动顺序 模块 Application类 端口 1 train-gateway GatewayApplication 8080 2 train-user UserApplication 8081 3 train-train TrainApplication 8082 4 train-ticket TicketApplication 8083 5 train-order OrderApplication 8084 6 train-pay PayApplication 8085

4. 运行配置

- 每个模块的启动类在 src/main/java/com/zhenyu/train/xxx/ 目录下
- 右键 Application 类 → Run 'xxxApplication'
- 或者点击类名旁边的绿色三角按钮
5. 注意事项

- 每个服务会占用一个终端，IDEA底部的 Services 面板可以统一管理所有运行中的服务
- 建议开启 Services 面板：菜单 View → Tool Windows → Services
- 在Services面板中可以一键启动/停止/重启所有服务
6. 验证

启动完成后，访问网关验证：

```
http://localhost:8090/api/user/register
```

注意：网关端口是 **8090**，路由前缀是 **/api/**

各服务接口地址：

| 服务 | 网关地址 | 直连地址 |
|------|---------|---------|
| 用户服务 | http://localhost:8090/api/user/... | http://localhost:8081/user/... |
| 车次服务 | http://localhost:8090/api/train/... | http://localhost:8082/train/... |
| 余票服务 | http://localhost:8090/api/ticket/... | http://localhost:8083/ticket/... |
| 订单服务 | http://localhost:8090/api/order/... | http://localhost:8084/order/... |
| 支付服务 | http://localhost:8090/api/pay/... | http://localhost:8085/pay/... |

访问Nacos控制台查看服务注册情况：

```
http://localhost:8848/nacos
```
用户名/密码：nacos / nacos
